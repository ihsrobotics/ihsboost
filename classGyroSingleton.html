Detailed Description

Singleton class who's job it is to accumulate gyro values.

Since this is a singleton class, use instance() to get a pointer to the GyroSingleton object; this pointer can be used to access the functionality of this class. Note: get_accumulator is equivalent to getting the current angle that the brain is at.
