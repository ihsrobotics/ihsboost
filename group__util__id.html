◆ get_config()

Config& get_config ( std::string  config_file = CONFIG_FILE)

Get the ihsboost config object holding all tunable ihsboost variables.

The tunable ihsboost variables are:

  • mean_gyro_val - the mean raw value returned by the gyro function
  • min_gyro_val - the min raw value returned by the gyro function
  • max_gyro_val - the max raw value returned by the gyro function
  • raw_to_360_degrees - multiplying accumulated gyro values by this should result in degrees
  • gyro_cw_multiplier - multiply by this number so that positive values represent CW rotation and negative values represent CCW rotation
  • gyro_updates_per_sec - default updates per second for gyro functions
  • gyro_correction_proportion - default correction for gyro_drive_straight
    See also
    gyro_drive_straight
  • gyro_min_speed - default min speed for gyro_turn_degrees_v2
    See also
    gyro_turn_degrees_v2
  • gyro_accel_per_sec - default accel for gyro functions
  • gyro_stop - whether or not to do a full stop after gyro movement functions
  • linear_accel - default linear acceleration to use in accelerate_linear; should always be positive
    See also
    accelerate_linear
  • sinusoidal_accel - default sinusoidal acceleration to use in accelerate_sinusoidal; should always be positive
    See also
    accelerate_sinusoidal
  • accelerate_updates_per_sec - default updates per second when accelerating
  • black - greater than this value is white, less than is black
  • line_follow_updates_per_sec - default updates per second when line following
  • line_follow_correction_proportion - default correction for line following
  • line_follow_linear_acceleration - default linear acceleration to use in line_follow_accelerate_linear
    See also
    line_follow_accelerate_linear
  • line_follow_sinusoidal_acceleration - default sinusoidal acceleration to use in line_follow_accelerate_sinusoidal
    See also
    line_follow_accelerate_sinusoidal
  • line_follow_stop - whether or not to default to a full stop after line following
  • align_updates_per_sec - default updates per second when aligning
  • align_stop - whether or not to default to a full stop after align functions
  • roomba_correction_proportion - default correction for the roomba
    See also
    encoder_drive_straight
  • roomba_updates_per_sec - default updates per second for the roomba
  • roomba_accel_per_sec - default acceleration for the roomba
  • roomba_min_speed - default min speed for the roomba
  • roomba_stop - whether or not to do a full stop after roomba movement functions
  • roomba_left_wheel_units - how far the left wheel moves, in cm/sec
    See also
    rotate
  • roomba_right_wheel_units - how far the right wheel moves, in cm/sec
    See also
    rotate
  • servo_movement_speed - default number of ticks to move a servo per second
    See also
    move_servo_slowly
  • servo_updates_per_sec - default updates per second when moving a servo
    See also
    move_servo_slowly
    Parameters
    config_filethe name of the config file
    Returns
    Config& - the config
