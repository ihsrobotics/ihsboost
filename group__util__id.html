◆ get_config()

Config& get_config ( std::string  config_file = CONFIG_FILE)

Get the ihsboost config object holding all tunable ihsboost variables.

The tunable ihsboost variables are:

  • mean_gyro_val - the mean raw value returned by the gyro function
  • min_gyro_val - the min raw value returned by the gyro function
  • max_gyro_val - the max raw value returned by the gyro function
  • raw_to_360_degrees - multiplying accumulated gyro values by this should result in degrees
  • gyro_cw_multiplier - multiply by this number so that positive values represent CW rotation and negative values represent CCW rotation

config_filethe name of the config file
Returns
Config& - the config
